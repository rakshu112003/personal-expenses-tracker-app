# personal-expenses-tracker-app
💰 Expense Tracker

A full-stack Expense Tracker application built using the MERN Stack. This application helps users manage their daily income and expenses, categorize transactions, and monitor spending through a simple and responsive interface.

📌 Features

- User Authentication (Login/Logout)
- Add Income & Expenses
- Edit and Delete Transactions
- Expense Categories
- Search and Filter Transactions
- Dashboard with Summary
- Responsive User Interface
- Secure REST API
- MongoDB Database Integration

🛠️ Tech Stack

Frontend

- React.js
- Vite
- JavaScript
- HTML5
- CSS3
- Axios
- React Router DOM

Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- REST API

🚀 Getting Started

Clone the Repository

git clone https://github.com/your-username/expense-tracker.git
cd expense-tracker

Frontend

cd frontend
npm install
npm run dev

Backend

cd backend
npm install
npm start
